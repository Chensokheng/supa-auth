# Installation

```sh copy
npx supaauth@latest add --component signup
```

This will do:

1. create `signup.tsx` file for you
2. It will install all dependencies by running these cmds

```sh
npx shadcn-ui@latest add form
npx shadcn-ui@latest add input
npm i react-icons
```

## Skip dependencies

If you have all dependencies you can skip it by do this

```sh copy
npx supaauth@latest add --component signup --skip dep
```
